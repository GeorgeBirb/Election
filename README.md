# Election
Blockchain Election Development with Solidity

!!DON'T FORGET TO ADD "node_modules" file!!!

* Open GANACHE 
* cd Election
* Ctrl+C 
* truffle compile
* truffle migrate --reset --network development 
* truffle development
* Election.deployed().then(function(instance) { app = instance; });
* Ctrl+C 
* npm run dev
